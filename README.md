# Chat bot

- to build for all platforms, run

```shell
npm run build:all
```

- to start web run

```shell
npm run dev
```

- to build for a mobile platform, you need to run

```shell
npm run gen:add-android
```

- to start the mobile work environment, run open:android

```shell
npm run open:android
```

## Stack

* TS
* React
* React-Router-Dom
* Redux-Toolkit
* RTK
* SCSS
* Vite
* Generate Scripts
* Eslint
* Prettier
* Axios
